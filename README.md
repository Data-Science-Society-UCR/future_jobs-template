## Project Overview 

As the job market shifts, especially in the engineering and computer science fields, it is important to know what skills to invest in and to see how those skills can empower you to become a strong candidate for large industries. The goal of this project is to highlight skills that frequently showed up in top tech companies in 2025. This will be done by analyzing the data collected from Kaggle. The data divides the 10,000 job postings by salary, location, accessibility, and skills. Overall, this project serves as a practical guide for anyone seeking to align their skills with the industry they intend to enter.
